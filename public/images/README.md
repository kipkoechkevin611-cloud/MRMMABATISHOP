# Images Directory

This directory contains images for the MRM Roofing & Steel website.

## Required Images

### Hero Background
- `hero-bg.jpg` - Background image for the hero section (recommended: 1920x1080px, roofing/construction theme)

### Product Images
- `box-profile.jpg` - Box Profile Mabati product image
- `corrugated.jpg` - Corrugated Mabati product image
- `versatile.jpg` - Versatile Mabati product image
- `romantile.jpg` - Romantile product image
- `euro-tile.jpg` - Euro Tile product image
- `zee-tile.jpg` - Zee Tile Mabati product image
- `dumuzaz.jpg` - Dumuzaz Mabati product image
- `plain-sheet.jpg` - Plain Sheet product image
- `flat-ridges.jpg` - Flat Ridges product image
- `glazed-ridges.jpg` - Glazed Ridges product image
- `valley-trays.jpg` - Valley Trays product image

### Placeholder
- `placeholder.jpg` - Fallback image when product images fail to load

## Image Specifications
- Recommended size: 800x600px for product images
- Format: JPG or PNG
- File size: Keep under 500KB for optimal performance
- Background: White or transparent for product images

## Adding Images
Simply place your image files in this directory with the exact filenames listed above.
The application will automatically load them.
